# Bundled icon attributions

Vendored third-party SVG glyphs in this directory, by source:

- **File-type glyphs** (`rust.svg`, `typescript.svg`, `python.svg`, …) — Seti UI
  ([jesseweed/seti-ui](https://github.com/jesseweed/seti-ui)), MIT. The glyphs
  behind Zed's default icon theme; see `docs/spec-explorer-icons.md`.
- **`git-branch.svg`** — Lucide
  ([lucide-icons/lucide](https://github.com/lucide-icons/lucide)), ISC. The
  activity rail's Git area icon (`docs/archive/spec-workspace-visibility-rail.md`).
  gpui-component's own `IconName` set is also Lucide.

Both licenses are permissive and GPL-3.0-compatible; the workspace ships under
GPL-3.0-or-later (root `LICENSE`).
